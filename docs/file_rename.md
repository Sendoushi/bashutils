# Bedrock: Utils > File: Rename

Module to rename files / folders.

### Config file parameters
```toml
[[rename]]
source = "" # required
destination = "" # required
ignore = ""
order = 0
env = ""
cmd = ""
sys = "all"
```

### Examples
Go under the [../test/example_build](../test/example_build) folder and check the `*.toml`.
