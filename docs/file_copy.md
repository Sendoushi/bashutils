# Bedrock: Utils > File: Copy

Module to copy files / folder.

### Config file parameters
```json
{
    "type": "copy",
    "data": [{
        "src": "<task_src_glob>",
        "dest": "<task_src_glob>"
    }]    
}
```

### Examples
Go under the [../test/examples/copy](../test/examples/copy) folder and check the `*.json`.
